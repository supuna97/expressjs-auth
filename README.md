# Expressjs Authentication REST APIs with JWT Token

## Project technologies/tools
- Node.js
- Express.js
- NODEMON
- JWT
- BCRYPTJS
- JOI
- DOTENV
- MYSQL2
- Postman

## Project setup
```
npm init
```

### Install dependencies
    
```
npm install express
```

```
npm install nodemon
```

```
npm install jsonwebtoken
```

```
npm install bcryptjs
```

```
npm install joi
```

```
npm install dotenv
```

```
npm install mysql2
```
