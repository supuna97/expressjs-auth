# Expressjs Authentication REST APIs with JWT Token

## Project technologies/tools
- Node.js
- Express.js
- NODEMON
- JWT
- BCRYPTJS
- JOI
- DOTENV
- MYSQL2
- Postman

## Project setup
```bash
npm install
```npm install express```
```npm install nodemon```
```npm install jsonwebtoken```
```npm install bcryptjs```
```npm install joi```
```npm install dotenv```
```npm install mysql2```

### Compiles and hot-reloads for development
```bash
nodemon app.js
```
